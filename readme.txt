== Description ==

**Classifieds Theme** professional classifieds plugin for Wordpress.

Turn a standard Wordpress blog into a powerful, feature rich classifieds website easily with our Classifieds Theme.

Classifieds Theme has been professionally designed and developed to be simple and easy to use! It takes seconds to download, activate and begin running your very own classified ad website.

== Installation ==

1. Unzip into your `/wp-content/themes/` directory. If you're uploading it make sure to upload
the top-level folder. Don't just upload all the php files and put them in `/wp-content/themes/`.
1. Activate the plugin through the 'Appearance' menu in WordPress

For upgrading please use check our website http://www.classifiedstheme.com/
If you have to upgrade manually simply repeat the installation steps.
